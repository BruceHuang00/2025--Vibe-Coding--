 2025-Vibe-Coding-黃詠信

### 1. 設計理念
我想創造一個結合「嚴謹邏輯」與「教育溫度」的氛圍。
作為資工系學生，網頁採用極簡的 IDE 淺色風格展現專業；
作為數學老師，我加入了珠心算與卡牌玩家的幽默元素，
讓這張數位名片不只是冰冷的代碼，更是我跨領域特質的縮影。

### 2. 關鍵提示詞 (Prompt)
我認為最成功的兩個指令是：
* 「我是一位資工大四生兼數學老師。請用程式 while 迴圈語法設計一個標語，展現『算無遺策』的個人特質，風格要極簡淺色系。」
* 「請在網頁中加入程式風格的幽默註解（如 //TODO 或 //DEBUG），並保留兩句核心標語以輪播方式呈現。」

### 3. 問題解決紀錄
* **標語跑版問題**：兩句標語長度不同導致切換時下方內容會跳動，我透過設定 CSS `min-height` 固定容器高度解決。
* **分離 CSS 挑戰**：老師要求 HTML 與 CSS 分開存檔，我重新調整代碼結構，並確保在 `index.html` 中正確連結 `style.css`。
* **響應式調整**：初次生成時手機版排版混亂，我請 AI 協助加入 Tailwind 的響應式設計，確保在各種裝置都能完美顯示。

### 4. 自我心得
透過與 Gemini 的多次對話迭代，學會如何精確描述需求（如：將標語轉為 while 迴圈語法），並解決了 CSS 分離與響應式排版等實作難題。
這不僅完成了期末作業，更體會到 AI 作為開發助手，能大幅縮短從「願景」到「產出」的距離。
